# Directive: Media Vault

## Goal
Create a `MediaVault` page for clients to dump images/videos.

## Features
- **Upload**: Drag-and-drop zone (`react-dropzone`).
- **Storage**: Direct upload to AWS S3 (presigned URLs) or Firebase Storage.
- **Organization**: Auto-sort into folders: 'Logos', 'Images', 'Documents'.
- **Visuals**: Thumbnail previews for images, generic icons for PDFs/Docs.
